## Simple CLI 21 card game ([Blackjack](https://en.wikipedia.org/wiki/Blackjack])) devloped using OOP (JavaScript)

## Installation

`git clone https://github.com/olgashi/21_Card_Game_OOP`

`cd 21_Card_Game_OOP`

`npm install`

## To run

`npm start`
